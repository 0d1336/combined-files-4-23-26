# Removing Overlapping Shapes from a PDF

Sometimes I run into a PDF with awkward geometry sitting on top of the content. Stray rectangles, overlapping SVG paths, the occasional plot artifact, that sort of thing really makes reading text difficult. 

Complicating things, I'm not really a people person. I like ordering food online and voting by mail, and the thought of bothering someone solely for the sake of reading text they sent me is just too much.

Anyway, my approach is to just do a bit of mopping up and thought I'd share how it's done. The included `recover.py` only works with this single input PDF but I'm publishing the code so anyone with some coding experience and the willingness to learn can do likewise in the future.
